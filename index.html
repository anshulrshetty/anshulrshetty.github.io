About me

I build the tools that turn data into lap time. My focus is vehicle dynamics and simulation: constructing lap time models, correlating them against real telemetry, and translating driver behaviour into concrete setup actions.

That thread runs through everything on this page - a dissertation simulating 2026-regulation Formula 1 cars in AVL VSM, a Python platform that fingerprints driving style from racing telemetry and proposes setup changes by Bayesian optimisation, and a patent-pending tyre pressure control system that closed the loop on real hardware.

Away from the desk: accredited Motorsport UK marshal, sim racer on a G923 rig, go-karting, and Formula 1 technical analysis.

Projects

MSc dissertation In progress

F1 Vehicle Dynamics Optimisation

Lap time simulation of 2026-regulation Formula 1 cars in AVL VSM, modelling active-aero X/Z mode switching and a 350 kW MGU-K torque overlay. Validated against real telemetry through a FastF1 Python pipeline, targeting under 2% lap time error across Monaco, Silverstone and Spa. A machine-learning layer classifies driving style and links it to setup direction.

<2% lap time error target 3 circuits validated k=5 cross-validation

AVL VSM · FastF1 · Python · XGBoost · K-means · SHAP

ACC Driver Fingerprinting & Setup Optimisation

End-to-end Python platform built on MoTeC exports from Assetto Corsa Competizione. It detects corners, extracts per-corner driver features - braking aggression, trail-braking overlap, throttle pickup, lateral G - and proposes the next setup change by Bayesian optimisation, tested one factor at a time on a G923 rig.

Per-corner driver fingerprinting Bayesian next-setup proposals OFAT test methodology

Python · MoTeC · Bayesian optimisation · Signal processing

Smart Tyre Pressure Control System

Closed-loop tyre pressure control from real-time sensor data, integrating a pneumatic valve with an ECU to hold pressures within ±0.3 bar. Led a team of four from concept to functional prototype; industry validated, 2nd place at Technova 2023-24 and a KSCST grant winner.

±0.3 bar control accuracy 2nd place Technova 2023-24 Industry validated
